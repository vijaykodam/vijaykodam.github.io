Claude Code CLI

Claude Code is Anthropic’s official command-line interface for Claude. It’s an agentic coding tool that lives in your terminal, understands your codebase, and helps you code faster through natural language commands.

Features

  • Agentic coding: Claude can read, write, and edit files autonomously
  • Terminal integration: Runs shell commands and understands output
  • Codebase awareness: Understands project structure and context
  • Multi-file editing: Can work across multiple files simultaneously
  • Git integration: Commits, creates PRs, and manages version control

Installation

npm install -g @anthropic-ai/claude-code
