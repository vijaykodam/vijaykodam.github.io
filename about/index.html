Vijay Kodam

This is Vijay Kodam‘s personal blog. He is currently working as a Principal Engineer Cloud Technology at Nokia, Finland.

He started developing software for 2G CDMA in his first job, then moved onto 3G 1x-EVDO RNC development. After that he wrote code to deploy 4G/5G Virtual Network Functions on Telco Cloud using Openstack, Mesos and Docker.

Currently he is managing cloud infrastructure using Kubernetes, Istio, Docker, CoreOS, Ambassador, and Harbor on bare metal and AWS. He dons several hats as a Cloud Architect, SRE, and system admin. He is a Certified Kubernetes Administrator, Certified Kubernetes Application Developer and AWS Certified Solutions Architect - Associate.

When he is not working in Cloud computing, he gets his hands dirty with IoT hobby projects.

The views expressed in this blog are entirely his personal views and not his employer’s.
