About Me

I’m Vijay Kodam, a seasoned technology professional with deep expertise in Amazon Web Services, cloud architecture, software development, and system design. I currently work as a Principal Engineer in Cloud Technology at Nokia, Finland.

My career journey has been a blend of deep telecom expertise and cutting-edge cloud computing skills. I began by developing software for 2G CDMA systems, progressed to building 3G 1x-EVDO RNC solutions, and then transitioned to deploying 4G/5G Virtual Network Functions (VNFs) on Telco Cloud using platforms like AWS, containers and Kubernetes. Over the years, I’ve expanded my focus to Amazon Web Service(AWS), Systems Design, Solution Architecture and Kubernetes, specializing in Amazon EKS to build scalable and resilient cloud-native solutions for telecom applications.

Currently, I’m leading and executing the end-to-end architecture and migration of a Private 5G platform into AWS. This involves designing, implementing, and managing solutions using Amazon EKS, various AWS services, Cassandra, Grafana, Prometheus, and containerized applications. My role encompasses Cloud Architecture, SRE, DevOps, GitOps, and Systems Design, enabling me to deliver robust, scalable, and high-performing cloud-native solutions from start to finish.

In my spare time, I delve into AI and Large Language Model (LLM) projects, which are documented in the Projects section of this blog.
