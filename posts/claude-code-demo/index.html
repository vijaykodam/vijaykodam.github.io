Here is my Day Zero hands-on demo of Claude Code using Claude 3.7 Sonnet. Both were released by Anthropic today.

Claude 3.7 Sonnet, is the first hybrid reasoning model on the market by Anthropic. They have also introduced a command line tool for agentic coding, Claude Code.

I have installed in my mac and tried agentic code and made a demo video for you.

Take a look and add in comments how are you planning to use this shiny new agentic code tool in your workflow?

It cost me $0.3273 for 1 minute usage of Anthropic API. It is costly for personal use. That is my personal feeling.

I would rather want this Claude code like agentic code tool to run locally and call a local llm running on my mac. Everything local, and secure solution. Of course, there will be some opensource tool out in the market in a month doing the same thing. Let’s hope so…
