Running Deepseek R1 locally

January 29, 2025 1-minute read
Cloud • GenAI • LLM
GenAI • LLM • OpenAI

Here is a video of Deepseek R1 running locally on my Macbook. Now that everyone is amazed by the low level of resource utilization and open weight model of Deepseek R1, installed 8B model locally.

With the help of ollama it is as simple as running a single command - “ollama run deepseek-r1:8b” You get your own private, local LLM running securely in your computer.
