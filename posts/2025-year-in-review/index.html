2025 Year in Review

December 30, 2025 2-minute read
Cloud • AWS • GenAI
AWS • LLM • GenAI • Kubernetes • Agents

If you had told me at the start of the year that I would speak at major Amazon Web Services (AWS) and Microsoft conferences in Europe, I would have called it unlikely.

2025 has been an amazing year for me. Not only I got the chance to speak at AWS Summit and Microsoft AgentCon and other conferences, I also got certified, completed GenAI courses, built MCP servers and AI agents and much more. I have spent countless hours after work this year to achieve those things.

Here are a few highlights for me in 2025:

  • Became AWS Community Builder for the first time
  • Passed AWS Certified AI Practitioner from Amazon Web Services (AWS)!
  • Created my own Kubernetes Read Only MCP server
  • Created my own AI Agent for Autonomous Troubleshooting K8s operations.
  • Delivered presentations at AWS Summit Zurich, Microsoft’s AgentCon and AWS Nordics Community Day!
  • Won two GenAI hackathons at AWS Summit Stockholm and another at AWS event in Helsinki!
  • Learnt a lot in #GenAI including “Building Agentic AI Applications with a Problem-First Approach”, “AI evals” and Google’s GenAI and AI Agent courses.
  • Built multiple AI projects. “LLM pricing” project is the latest one.
  • Trained a neural network for my Le Robot SO-101 arm.

All throughout the year along my journey, many people helped me, inspired me and guided me. Thanks to all of you!
